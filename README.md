# Aurore et le Trésore d'Aurore

Un jeu style RPG en 2D, dans lequel nous incarnons Aurore une petetie aventurière qui est à la recherche d'un grand trésor. Pour y arriver, elle devra franchir divers obstacles que ce soit des monstres à combattre ou des portes à débloquer afin d'atteindre son but


## Installation:

### Pré-requis:
    Python 3.10 +

### Dépendances:
    Pygame : pip install pygame

## Utilisation
    
Lancer le fichier "main.py"

### Contrôles:
    Déplacer à gauche : ←
    Déplacer à droite : →
    Déplacer en haut : ↑
    Déplacer en bas : ↓
    Pause : Esc
    L'inventaire : I
    Prendre un objet: E
    Utiliser un objet dans l’inventaire : E 
    Equiper une arme dans l’inventaire : E 
    Déséquiper une arme dans l’inventaire : E 
    Attaquer : X
